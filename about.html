About



The Minecraft Property Encyclopedia is an open source project that is actively being made by a team of contributers.


The information in this encyclopedia is accurate as of Minecraft Java Edition version 1.18.


You can check out the Github Page for info about contributing to this project.
For contact and further information be sure to visit the designated channel in the MelonTech discord server.
